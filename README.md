# Momo——Build your mastersheet

基于 Key 和匹配值的 Excel 批量处理工具集，支持多文件并发处理。

## 主要功能

### 填表工具
- **Master到小表**：从主表批量更新多个小表文件
- **小表到Master**：从多个小表收集数据更新到主表
- **多列更新**：同时更新多个列的内容

### 辅助工具
- **列清空**：批量清空指定列的内容
- **兼容性处理**：处理Excel文件兼容性问题
- **深度替换**：用源文件夹中的文件替换目标文件夹中的同名文件

## 注意事项
- 处理前请备份重要文件
- 确保目标文件未被其他程序占用
- 建议在处理大量文件前先小批量测试
